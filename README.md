## Features

- Minimal
- TypeScript
- [esbuild](https://github.com/evanw/esbuild) to build
- [Miniflare](https://github.com/cloudflare/miniflare) to develop
- [Jest](https://jestjs.io/ja/) to test
- Fast
- Live reload

## Usage

Initialize

```sh
$ wrangler generate my-app https://github.com/yusukebe/hono-minimal
```

Install

```sh
$ yarn install

```

Develop

```sh
$ yarn dev
```

Test

```
$ yarn test
```

Deploy

```
$ yarn deploy
```
